📌 종합 인사이트

  • 분양 럭셔리 사이트의 정점 — anadd.co.kr/apt가 9.8점으로 가장 풍부. 9개 키프레임 직접 추출 가능한 수준의 정밀한 motion.css 설계.
  • 한자 헤리티지 위계가 차별화 핵심 — prugio summit과 anadd/apt가 타이포 10점. 영문 세리프(Trajan/Cormorant) + 한글 명조 + 한자 디스플레이 트리플 페어링.
  • 4-layer 패럴랙스의 정수 — 르엘 리버파크는 vis-base / vis-line × 2 / ball / ball-shadow 5계층 분리로 럭셔리 깊이감 구현.
  • 비디오 BG는 양날의 검 — saem.space는 비디오로 분위기를 압도하지만 모바일 fallback이 모호. 정적 fallback 이미지 명시 필요.
  • fullPage.js는 여전히 유효 — 2026년에도 fullPage 패턴이 럭셔리 분양에 효과적. 다만 모바일 mousewheel/터치 호환성 별도 정책 필수.
  • 마케팅 메인은 럭셔리 차별화 약함 — ssyapt 메인 같은 통합 마케팅 게이트는 단지별 디테일 사이트(/apt 같은)로 깊이 분기하는 구조가 표준.
  • 가장 약한 축은 모바일 — 8개 사이트 공통적으로 모바일 fallback 정책이 데스크톱 대비 약함. Gyroscope / DeviceOrientation 활용은 르엘만 가능성 시사.
  • 커스텀 키프레임 vs 라이브러리 — 커스텀 motion.css(anadd/apt)가 라이브러리 의존(ssyapt)보다 정체성 차별화에 유리.
