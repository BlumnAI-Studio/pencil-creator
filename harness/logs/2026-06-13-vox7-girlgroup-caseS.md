## 3. Phase별 요약

- **Gather**: 영상 다운로드(video-motion-analysis prepare) → overview/dense 콘택트 시트로 7인 의상·헤어 식별. 영상이 토크 위주라 개별 노래 제스처 대신 **리듬 스탠드 댄스**를 핵심 액션으로 채택(사용자 지시).
- **Pilot**: vox7-1 풀 스코프(컨셉+idle6+dance8) → 93/A 확인 후 게이트 통과 → 나머지 6명 배치.
- **Batch**: gpt-image-2 배치 드라이버(`tmp/gen_vox7_batch.py`) 백그라운드 실행. vox7-2 1차 누락(컨셉 일시 실패 연쇄) → **재개 가능 드라이버 재실행으로 누락분만 자동 보충**.
- **Post**: 멤버별 컨셉에서 48색 팔레트 추출 → idle/dance 후처리(`tmp/post_vox7.py`).
- **Verify**: 7명 전원 93/A.

## 4. 핵심 발견

1. **모던 색상엔 per-character 팔레트 필수**: 글로벌 다크판타지 어스톤 팔레트를 강제하면
   교복 색(네이비/레드/옐로/데님)이 왜곡된다. 컨셉에서 48색 적응 팔레트를 추출해 적용하니
   드리프트 0.0 — 멤버 고유색 완벽 보존. (vocal-ex의 글로벌 팔레트와 대비되는 신규 패턴)
2. **스탠드 댄스 8프레임 루프**: 중앙→좌 스텝/암스윙→좌 포즈→복귀 바운스→우 스텝→우 포즈→
   숄더 바운스→홉 클랩. 마이크 한 손 + 반대 팔 리듬 제스처로 idol 제자리 안무가 자연스럽게 읽힌다.
3. **배치 실패 복원력**: 한 멤버 전체 실패(연쇄)도 skip-exists 드라이버 재실행으로 무손실 복구.
4. **gpt-image-2 7인 일관성**: 멤버별 단일 컨셉을 reference로 고정 → 14프레임 내내 의상/헤어 유지.
