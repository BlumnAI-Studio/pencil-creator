# 개발컨셉
우선 저작권이 없는 midi 파일을 획득합니다. ( 오스스트레이션)
(최소5개 서칭)
미디 파일을 로드하면 악기의 연주구간을 파악해
해당 악기가 연주될때 캐릭터가 재생합니다.
악기가 새롭게 등장하면 씬에서 적절한 위치를 잡습니다.
동적으로 추가가되지만 제거는 되지 않고 음악이 나올때 까지 스탠바이합니다.
배경은 제미나이로 연주 백그라우드 무대테마를 3개정도 선택합니다.
배경을 파악해 연주자가 추가되면 포지션할 위치를 설정합니다.
콤보박스에 음악을 선택 플레이하면 재생됩니다.

새로운 midi파일이 추가될수 있으니 이 방법은 하네스 전문가를 추가하고 지식을 보유합니다.
